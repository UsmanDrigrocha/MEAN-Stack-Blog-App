# MEAN Stack Blogging App

## To Run Frontend 
```cd frontend ``` <br><br>
``` npm install``` <br><br>
``` ng serve ``` <br><br>


## To Run Backend

 create ``` .env``` file and enter the following <br><br>
``` JWT_SECRET_KEY =   PORT=    DB_VAR=```<br><br>
``` cd server ```<br><br>
``` npm install ```<br><br>
```npm run dev ```<br><br>
